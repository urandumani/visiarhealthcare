# VisiarHealthcare

A full-stack healthcare web application with **Next.js frontend** and **.NET backend**, containerized with **Docker Compose** for easy development and deployment.

## Prerequisites

- [Docker](https://www.docker.com/get-started) & [Docker Compose](https://docs.docker.com/compose/)

- ## Setup & Run (Docker)

1. Clone the repository:

```bash
git clone https://github.com/urandumani/visiarhealthcare.git
```

2. Run the app with docker:

```bash
docker-compose up
```

3. Open the app in your browser:
```bash
localhost:3000
```
